**Variables:**
- `INSTALL_DIR="${CLAUDE_RESTART_INSTALL_DIR:-$HOME/.local/bin}"` (env var override for testing)
- `ZSHRC="${CLAUDE_RESTART_ZSHRC:-$HOME/.zshrc}"` (env var override for testing)
- `SCRIPT_DIR="$(cd "$(dirname "$0")" && pwd)"` (to find source scripts)
- Sentinel markers: `# >>> claude-restart >>>` and `# <<< claude-restart <<<`

**Install mode (default, no flags or explicit `--install`):**

1. Create `$INSTALL_DIR` if it doesn't exist (`mkdir -p`)
2. Copy `$SCRIPT_DIR/claude-wrapper` to `$INSTALL_DIR/claude-wrapper` and `chmod +x`
3. Copy `$SCRIPT_DIR/claude-restart` to `$INSTALL_DIR/claude-restart` and `chmod +x`
4. Check if `.zshrc` already contains the sentinel `# >>> claude-restart >>>`:
   - If YES: print "claude-restart: already configured in $ZSHRC (skipping)" and skip step 5
   - If NO: proceed to step 5
5. Append this block to `$ZSHRC`:

```
# >>> claude-restart >>>
export CLAUDE_RESTART_DEFAULT_OPTS="--dangerously-skip-permissions --channels plugin:telegram@claude-plugins-official"
export PATH="$INSTALL_DIR:$PATH"
claude-restart() {
    if [[ $# -gt 0 ]]; then
        "$INSTALL_DIR/claude-wrapper" "$@"
    else
        # shellcheck disable=SC2086
        "$INSTALL_DIR/claude-wrapper" $CLAUDE_RESTART_DEFAULT_OPTS
    fi
}
# <<< claude-restart <<<
```

Note: The `$INSTALL_DIR` references inside the appended block must be expanded at install time (use double quotes in the heredoc or variable substitution), so the actual absolute path is written to `.zshrc`, not the variable name.

The `CLAUDE_RESTART_DEFAULT_OPTS` reference inside the function body must NOT be expanded at install time — it must remain as `$CLAUDE_RESTART_DEFAULT_OPTS` so it reads the env var at runtime. Use single-quoted or escaped syntax for that line.

6. Print success messages:
   - "Installed claude-wrapper to $INSTALL_DIR/claude-wrapper"
   - "Installed claude-restart to $INSTALL_DIR/claude-restart"
   - "Added shell function to $ZSHRC"
   - "Run 'source $ZSHRC' or open a new terminal to start using claude-restart"

**Uninstall mode (`--uninstall` flag):**

1. Remove `$INSTALL_DIR/claude-wrapper` and `$INSTALL_DIR/claude-restart` (with `-f`, no error if missing)
2. If `$ZSHRC` contains sentinel `# >>> claude-restart >>>`:
   - Use `sed` to delete from `# >>> claude-restart >>>` through `# <<< claude-restart <<<` (inclusive)
   - Print "Removed claude-restart configuration from $ZSHRC"
3. If `$ZSHRC` does NOT contain sentinel: print "claude-restart: no configuration found in $ZSHRC"
4. Print "Uninstalled claude-restart from $INSTALL_DIR"

**Help mode (`--help` flag):**
Print usage: `Usage: install.sh [--install | --uninstall | --help]`

Make the script executable: `chmod +x bin/install.sh`
