```bash
usage() {
    echo "Usage: claude-service {start|stop|restart|status|logs|watchdog|heartbeat}"
    echo ""
    echo "Commands:"
    echo "  start      Start the claude service"
    echo "  stop       Stop the claude service"

    echo "  status     Show service status"
    echo "  logs       Follow service logs (journalctl)"
    echo "  watchdog   Show watchdog timer status"
    echo "  heartbeat  Show recent heartbeat log entries"
}
```

2. Add two new cases in the case statement, before the `--help|-h|""` line:

```bash
    watchdog)
        systemctl --user status claude-watchdog.timer
        ;;
    heartbeat)
        journalctl --user -u claude --grep="heartbeat sent" --no-pager -n 20
        ;;
```

The `watchdog` subcommand shows the timer status (when it last fired, when it fires next).
The `heartbeat` subcommand shows the 20 most recent heartbeat log entries from journald, filtering by the "heartbeat sent" string that the wrapper outputs to stderr.
